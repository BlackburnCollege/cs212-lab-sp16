jflac is an audio library that allows java to handle flac file types

You can reference the jlayer API documentation at http://jflac.sourceforge.net/apidocs/

I: Adding the jar file
-----------------------

1. Begin by right-clicking your homework repo in netbeans

2. Select Libraries on the left side of the window

3. Click the "Add JAR/Folder" button on the right side of the screen

4. Navigate to the lab repo's documents folder on your flashdrive (e.g. D:\cs212-lab\cs212-lab-sp16\Documents\Project1)

5. Select the jflac-1.3-levan.jar file and hit open

6. Press OK

II: Importing
--------------
In your java file you will need to add the following packages:

java.io
java.util.logging.Level;
java.util.logging.Logger;
javax.sound.sampled.LineUnavailableException;
org.kc7bfi.jflac.apps.Player;

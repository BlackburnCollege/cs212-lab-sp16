Tinysound is an audio library that allows java to handle wav and ogg vorbis file types

You can reference the Tinysound API documentation at http://finnkuusisto.github.io/TinySound/doc/

I: Adding the jar file
-----------------------

1. Begin by right-clicking your homework repo in netbeans

2. Select Libraries on the left side of the window

3. Click the "Add JAR/Folder" button on the right side of the screen

4. Navigate to the lab repo's documents folder on your flashdrive (e.g. D:\cs212-lab\cs212-lab-sp16\Documents\Project1)

5. Select the tinysound-1.1.1-levan.jar file and hit open

6. Press OK

II: Importing
--------------
In your java file you will need to import the following packages:

java.io.*;
kuusisto.tinysound.Music;
kuusisto.tinysound.TinySound;
